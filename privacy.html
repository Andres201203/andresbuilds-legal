Privacy Policy

Last updated: March 22, 2026

1. Introduction

This privacy policy describes how AndresBuilds Publisher, operated by Andres Escobedo Lara ("we", "us"), handles information. This is an internal content publishing tool — it is not a consumer-facing application.

2. Information We Collect

This application does not collect personal information from end users. The only data processed is:

  • OAuth tokens: Used to authenticate with TikTok and YouTube APIs on behalf of the account owner.
  • Video content metadata: Titles, descriptions, and scheduling data for content to be published.

3. How We Use Information

All data is used exclusively to publish video content to the authenticated social media accounts. No data is shared with, sold to, or disclosed to third parties.

4. Data Storage

OAuth tokens are stored securely in environment variables on a private server. Video files are stored temporarily during the upload process and are not retained after publishing.

5. Third-Party APIs

This application interacts with:

  • TikTok Content Posting API — governed by TikTok's Privacy Policy
  • YouTube Data API — governed by Google's Privacy Policy

6. Data Deletion

As this application does not collect user data, there is no personal data to delete. The account owner may revoke API access at any time through the respective platform settings.
